# 📦 Pânico no Galpão (Logistics Panic)

Um jogo de lógica e agilidade inspirado em Tetris, com foco em organização logística e gerenciamento de crise cromática. 

Este projeto foi criado para fins de **estudo e diversão**, explorando mecânicas de movimentação de blocos e lógica de colisão.

## 🕹️ O Jogo
No caos de um centro de distribuição, as caixas não param de chegar! Sua missão é mover e organizar as caixas da mesma cor para completar linhas e liberar espaço no galpão antes que o estoque lote e o pânico se instale.

### Mecânicas principais:
- **Movimentação:** Mover caixas para os lados.
- **Match de Cores:** Alinhar cores iguais para pontuar.
- **Limpeza de Linha:** Completar uma linha horizontal remove os blocos (estilo Tetris).
- **Dificuldade Progressiva:** A velocidade das esteiras aumenta conforme você avança.

## 🛠️ Tecnologias
- **Engine:** Unity 6.x (ou sua versão)
- **Linguagem:** C#
- **Render Pipeline:** URP (se estiver usando) ou Built-in
